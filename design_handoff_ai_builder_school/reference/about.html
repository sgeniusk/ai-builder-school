About

뉴스가 아닌
산출물을 쌓는다.

AI 열풍은 강하지만 "뭘 공부해야 일에 쓰이는가?"는 여전히 모호합니다. 우리는 매 주 하나의 산출물을 남기는 학교입니다.

Mission

AI를 쓰는 사람에서,
AI로 만드는 사람으로.

문제를 이해하고 · 최소 개념을 배우고 · Codex/Claude Code에게 미션을 내리고 · 빌드 · 검증 · 배포 · 회고. 이 7단계 리듬 자체가 우리의 제품입니다.

Why

왜 또 다른 학교인가.

기존 강의는 뷰어를 만들고, 뉴스레터는 유행을 알려줄 뿐 작동하는 것은 남기지 않습니다. 우리는 매 레슨마다 빌더로서의 산출물을 남기고, 모든 콘텐츠는 한국 직장의 맥락으로 씁니다.

Principles

4가지 원칙

우리가 모든 레슨을 설계할 때 지키는 운영 체제.

01

빌드 > 소비

매 레슨이 산출물로 끝난다. 완료 체크가 아니라 작동하는 결과물이 끝낸다.

02

도구 중립

Codex, Claude Code, Cursor Agent — 어떤 것을 써도 동작하도록 미션을 설계한다.

03

검증 먼저

환각 · 과의존 · 프롬프트 인젝션을 학습자가 직접 확인한다.

04

한국어 중심

번역체가 아니다. 한국 직장의 맥락으로 예시를 쓴다.

FAQ

자주 묻는 질문

개발자가 아니어도 되나요?
네. 트랙 5개 중 3개(직장인 / 기획자 / 크리에이터)는 비개발자를 위해 설계되었습니다.
어떤 AI 도구가 필요하나요?
Codex CLI, Claude Code, Cursor Agent 중 하나. 레슨은 도구 중립으로 작성되어 있습니다.
강의처럼 영상으로 보나요?
아닙니다. 7단계 러닝 루프를 따라 직접 빌드합니다. 매 주 산출물이 남습니다.
8주면 충분한가요?
MVP 경로(Phase 1~7 + Capstone)는 8주입니다. 트랙에 따라 6~10주로 조정됩니다.
레슨을 직접 추가할 수 있나요?
네. 비개발자도 src/content/lessons.ts에 객체를 추가하면 라우트가 자동 생성됩니다.
유료인가요?
MVP v0.2 기준으로 콘텐츠는 오픈. 커뮤니티 · 평가 배지는 이후 버전에서 제공 예정입니다.
