# 대시보드 서술 자동화 가이드

KPI 숫자를 신뢰할 수 있는 서술로 변환하는 워크플로. 사실과 가설을 분리하고, AI 코멘트를 QA한다.

## 이 폴더의 파일

| 파일 | 용도 |
|------|------|
| `narrative-prompt.md` | 사실/해석 분리 서술 생성 프롬프트 |
| `narrative-qa-checklist.md` | AI 코멘트 제출 전 QA 체크리스트 |

## 사용 순서

1. `narrative-prompt.md` 의 `[KPI 데이터]` 칸에 이번 달 수치 입력
2. `[청중]` 칸에 경영진 / 팀 리뷰 중 선택
3. AI 서술 생성
4. `narrative-qa-checklist.md` 로 1분 검토
5. 근거 없는 문장 삭제 또는 (가설) 표시 후 보고서 삽입

## 핵심

수치 계산은 AI가 틀릴 수 있다. 퍼센트 변화율은 항상 원본 데이터와 대조한다.
