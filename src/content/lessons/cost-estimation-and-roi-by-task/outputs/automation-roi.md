# 자동화 ROI 시트

> 이 파일은 AI Builder School · Stage 4 "이 자동화, 돈이 될까 — 업무별 비용과 ROI" 에서 제공되는 출발 템플릿입니다.
> 당신이 실제로 굴리는 자동화에 맞춰 자유롭게 수정하세요.

## 1. 자동화 목록

지금 굴리고 있거나 만들 예정인 자동화를 한 줄씩 적는다.

| B | | | |

## 2. 비용 추정 표

비용은 세 덩어리 — 도구 요금 + 만든 시간 + 유지 시간.

| # | 도구 요금 (월) | 만든 시간 (시간) | 유지 시간 (시간/주) | 비고 |
| --- | --- | --- | --- | --- |
| A | ₩ | h | h | 예: 처음 세팅 8h, 매주 점검 0.5h |
| B | ₩ | h | h | |
| C | ₩ | h | h | |

> 시간을 돈으로 환산할 때 쓸 내 시급: ₩______ /시간

## 3. 절약 추정 표

절약 = 매주 아낀 시간 × 시급. 도구 요금이 줄어든 게 있으면 함께 적는다.

| # | 매주 아낀 시간 (시간) | 시간 절약액 (월) | 기타 절약 (월) | 월 절약 합계 |
| --- | --- | --- | --- | --- |
| A | h | ₩ | ₩ | ₩ |
| B | h | ₩ | ₩ | ₩ |
| C | h | ₩ | ₩ | ₩ |

> 월 시간 절약액 = 매주 아낀 시간 × 시급 × 4.3주

## 4. ROI · 손익분기

ROI = 월 절약 ÷ 월 비용. 손익분기 = 만든 시간을 회수하는 데 걸리는 주.

| # | 월 비용 | 월 절약 | ROI (절약÷비용) | 손익분기 (주) |
| --- | --- | --- | --- | --- |
| A | ₩ | ₩ | | 주 |
| B | ₩ | ₩ | | 주 |
| C | ₩ | ₩ | | 주 |

> ROI 1 미만 = 아직 손해 · 손익분기 26주(반년) 초과 = 재검토 신호

## 5. 결정 — 더 투자 / 유지 / 접을 것

ROI와 손익분기를 보고 자동화마다 한 가지를 고른다.

| # | 결정 (더 투자 / 유지 / 접기) | 이유 한 줄 | 다음 액션 |
| --- | --- | --- | --- |
| A | | 예: ROI 3.2 — 절약 더 키울 여지 있음 | |
| B | | | |
| C | | | |

- **더 투자** — ROI가 높고 절약을 더 키울 수 있는 자동화
- **유지** — 본전은 넘었고 손 안 가는 자동화, 그대로 둔다
- **접기** — 손익분기가 너무 길거나 ROI가 1을 못 넘는 자동화
