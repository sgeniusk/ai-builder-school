# 터미널 치트시트

터미널 첫날 레슨 산출물. `commands.md`와 함께 쓴다.

---

## 오늘 실습한 것

아래 체크리스트를 완료하면 이 레슨이 끝난다.

- [ ] 터미널(또는 PowerShell)을 열었다
- [ ] `pwd` — 현재 위치를 확인했다
- [ ] `cd ~` — 홈으로 이동했다
- [ ] `ls` — 폴더 목록을 봤다
- [ ] `mkdir my-ai-builder-lab` — 실습 폴더를 만들었다
- [ ] `cd my-ai-builder-lab` — 폴더 안으로 들어갔다
- [ ] `touch README.md` — 빈 파일을 만들었다
- [ ] `ls -la` — 파일이 생성된 것을 확인했다

---

## 막혔을 때

**에러 메시지가 떴다.**
→ 에러 메시지 전체를 AI에게 붙여넣고 "Mac 터미널에서 이 에러가 떴어요. 어떻게 해결하나요?" 라고 물어본다.

**어디 있는지 모르겠다.**
→ `cd ~` 치면 항상 홈으로 돌아온다.

**명령이 안 끝나고 계속 기다린다.**
→ `Ctrl+C` 로 중단한다.

---

## 자주 쓰는 단축키

| 단축키 | 기능 |
|--------|------|
| `Tab` | 폴더·파일 이름 자동완성 |
| `↑` | 이전 명령어 불러오기 |
| `Ctrl+C` | 실행 중인 명령 중단 |
| `Ctrl+L` | 화면 지우기 |

더 많은 명령어는 `commands.md` 참고.
