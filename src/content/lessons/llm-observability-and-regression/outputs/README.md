# LLM 관측 시스템 가이드

프롬프트 변경 전후를 비교하는 회귀 테스트와 LLM 출력 품질 추적 최소 시스템.

## 이 폴더의 파일

| 파일 | 용도 |
|------|------|
| `log-schema.md` | LLM 호출 로그 필드 정의 |
| `golden-set.md` | 회귀 테스트 기준 입출력 쌍 |

## 사용 순서

1. `log-schema.md` 를 기반으로 LLM 호출마다 로그 기록
2. 프롬프트 변경 전 `golden-set.md` 에 현재 기대값 기록
3. 변경 후 골든 셋 재실행 → 달라진 케이스 확인
4. 실패 케이스 분석 → 의도적 변경인지 드리프트인지 판단
5. 드리프트면 프롬프트 롤백 또는 골든 셋 업데이트

## 핵심

`prompt_version` 없이는 "언제 달라진 건지" 알 수 없다.
