## 같아 보이는데, 다 다르다

ChatGPT를 쓰고 있다. 옆 동료는 Claude를 쓴다. 친구는 Cursor에서 코딩하고, 누군가는 Perplexity로 검색한다. 그리고 다들 매월 $20씩 결제하고 있다.

처음 보면 "다 비슷한 AI 서비스 아닌가"라고 생각한다. 그런데 자세히 보면 두 축으로 갈린다. **누가 모델을 만들었는가**, 그리고 **어떻게 돈을 내는가**.

이 두 축의 지도가 손에 있어야 — 매월 $80을 4개 서비스에 흩뿌리지 않고 — 내 작업에 맞는 도구를 고를 수 있다.

## 축 1 — 자체 모델 vs API 통합

**자체 모델 서비스** — 자기 회사가 만든 모델 + 자기 UI

| 서비스 | 모델 | 만든 곳 |
|---|---|---|
| ChatGPT | GPT-5.5 등 | OpenAI |
| Claude | Claude Opus·Sonnet·Haiku | Anthropic |
| Gemini | Gemini 3.5 Flash·Pro 등 | Google |

이 셋은 모델과 UI가 같은 회사에서 만든다. 모델이 업그레이드되면 UI도 따라간다. 안정성·일관성이 강하다.

**API 통합 서비스** — 여러 회사의 모델을 골라 쓰는 UI

| 서비스 | 무엇을 하나 | 어떤 모델을 쓰나 |
|---|---|---|
| Cursor | AI 코딩 IDE | GPT-5.5 · Claude · Gemini 다 |
| Perplexity | AI 검색·연구 | GPT·Claude·자체 모델 |
| Lindy | 비즈니스 자동화 에이전트 | OpenAI·Anthropic 모델 라우팅 |
| v0 | UI 코드 생성 | GPT-5.5 + 자체 fine-tune |
| Bolt.new | 풀스택 앱 생성 | Claude + 도구 호출 |

통합 서비스는 작업별로 최적 모델을 골라 쓸 수 있다. 단점은 추상화 계층이 한 겹 더 있어서 디버깅이 어렵고, 여러 회사의 비용이 합산된다.

**경험 법칙** — 모델 자체의 성능을 비교하고 싶다면 자체 모델 서비스. 특정 작업(코딩·검색·자동화)에 빠르게 결과를 내고 싶다면 통합 서비스.

## 축 2 — 구독제 vs API

같은 회사가 보통 둘 다 제공한다. 사용 모드가 다르다.

| | 구독제 (예 Plus·Pro·Advanced) | API |
|---|---|---|
| 과금 | 월정액 ($20·$100·$200) | 사용한 토큰만큼 |
| 인터페이스 | 브라우저·앱 채팅 UI | 내 코드에서 호출 |
| 한도 | 일·주별 사용 제한 (모델·기능별) | spending cap만 걸면 무한 |
| 강점 | 일상 사용 · 즉시 시작 · 멀티모달 UI | 자동화 · 시스템 빌드 · 대량 호출 |
| 약점 | 한도 도달하면 강제 대기 | 루프 폭주 시 청구서 폭탄 |
| 권장 시점 | 직접 대화·검토가 잦은 작업 | 무인 자동화·배치·앱 통합 |

**자주 혼동되는 것** — "ChatGPT Plus = ChatGPT API". 둘은 완전 분리된 시스템이다. Plus는 브라우저에서만 쓸 수 있고, API는 코드에서만 호출된다. 결제도 따로다.

## 3사 한 줄 차이 (2026년 상반기 기준)

> 휘발성 정보 — 6개월마다 바뀐다. 자세한 흐름은 특강 ["프런티어 AI 지형 2026"](/specials/frontier-ai-landscape-2026-05)에서.

- **OpenAI (ChatGPT · GPT-5.5)** — 에이전트 런타임에 베팅. ChatGPT iOS에서 Codex 원격 세션. 1M 컨텍스트.
- **Anthropic (Claude · Opus 4.7)** — 신뢰 가능한 추론·코딩. 어려운 SWE 작업·고해상도 비전. 가격 부담은 큼.
- **Google (Gemini · 3.5 Flash·Spark)** — Gmail·Drive·Calendar 통합 에이전트. Flash가 빠르고 싸고 좋은 모델 전략.

세 회사가 같은 방향(에이전트형)으로 수렴하지만 무게중심은 각각 다르다. **하나에 종속되지 말고 흐름을 알자**가 핵심.

## 그래서 오늘 할 일

내 작업에 맞는 서비스 지도를 한 장 만든다.

**작업 1 — 지금 쓰는 서비스 분류 (5분)**

지금 결제 중이거나 정기 사용 중인 AI 서비스를 모두 적기. 각각 표시

- 자체 모델 / 통합
- 구독제 / API / 무료 한도

**작업 2 — 내 주간 작업 5개를 분류 (15분)**

`service-rule.md` 한 장 만들기

```
| 작업 | 빈도 | 구독제로 충분 | API가 필요 | 추천 서비스 |
|---|---|---|---|---|
| 회의록 요약 | 매일 | ✓ |   | Claude Pro |
| 보고서 초안 | 주 2회 | ✓ |   | ChatGPT Plus |
| 사내 위키 자동 답변 봇 |   | ✓ (배포) | Anthropic API |
| 매일 5개 이메일 자동 분류 |   | ✓ (배포) | OpenAI API |
| ...
```

**작업 3 — 통합 서비스 1개 무료 체험 (10분)**

Cursor·Perplexity·Lindy·v0·Bolt 중 하나 무료 가입 → "여기서는 어떤 모델을 어떻게 쓰나" 한 줄로 관찰.

## 흔한 함정

- **모든 서비스 구독 신드롬** — Claude Pro + ChatGPT Plus + Gemini Advanced + Cursor + ... 매월 $80~$120. "한 달만 다 써보고 정하자"가 1년 가는 흔한 패턴.
- **API로 자동화 시작했는데 루프 폭주** — 무한 루프에 빠진 에이전트가 하루 만에 청구서 $500. spending cap 없이는 시작하지 말 것.
- **"Plus면 API도 무료"** 오해 — 둘은 별개. 헷갈리면 같은 회사 docs 다시 확인.
