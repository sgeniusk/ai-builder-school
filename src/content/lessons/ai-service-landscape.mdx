## 다 비슷해 보이는데, 고르는 기준이 있어요

ChatGPT를 쓰는데 옆 동료는 Claude를 써요. 친구는 Cursor에서 코딩하고, 누군가는 Perplexity로 검색하죠. 다들 매월 $20씩 결제하고 있고요.

처음엔 "다 비슷한 AI 아닌가" 싶어요. 그런데 한 발 물러서 보면, 고를 때 던질 질문은 사실 하나예요. **이건 모델을 직접 만든 회사의 도구인가, 아니면 남의 모델을 골라 쓰는 도구인가.** 이 한 줄만 잡아도 매월 $80을 네 곳에 흩뿌리는 일이 줄어들어요.

## 고르기 — 자체 모델이냐, 통합이냐

**자체 모델 서비스**는 자기 회사가 만든 모델에 자기 UI를 입힌 도구예요.

| 서비스 | 모델 | 만든 곳 |
|---|---|---|
| ChatGPT | GPT-5.5 등 | OpenAI |
| Claude | Opus·Sonnet·Haiku | Anthropic |
| Gemini | 3.5 Flash·Pro 등 | Google |

모델과 UI가 한 회사에서 나오니 **안정성과 일관성**이 강점이에요. 대신 모델을 갈아탈 자유는 적죠.

**통합 서비스**는 여러 회사의 모델을 골라 쓰는 UI예요. Cursor(코딩), Perplexity(검색), v0·Bolt(앱 생성)처럼 **특정 작업에 특화**돼 있어요. 작업별로 최적 모델을 골라 주는 게 강점이고, 대신 한 겹 더 추상화돼 있어 디버깅이 까다롭고 여러 회사 비용이 합산돼요.

> 경험 법칙 — 모델 자체를 비교하고 싶으면 자체 모델 서비스, 특정 작업에 빠르게 결과를 내고 싶으면 통합 서비스.

여기에 결제 방식이 한 겹 더 겹쳐요. 같은 회사라도 **구독제**(월정액 채팅 UI, 일상용)와 **API**(토큰 과금, 코드에서 호출, 자동화용)는 완전히 별개 시스템이에요. "Plus면 API도 공짜겠지"는 가장 흔한 오해죠 — 결제도 한도도 따로예요. 실제 계정·비용 세팅은 바로 다음 '계정·비용' 레슨에서 손에 잡히게 다룰게요.

## 익히기 — 어느 걸 골라도 공통 6기능

도구를 정했으면, 이제 그 속을 열어 볼 차례예요. 다행히 챗봇들은 **이름만 다를 뿐 거의 같은 6개 기능**을 갖고 있어요. 이 6개만 잡으면 어떤 챗봇으로 옮겨가도 금방 적응해요.

| 기능 | 무엇인가 | Claude / ChatGPT / Gemini |
|---|---|---|
| **작업공간** | 파일·지시문·대화를 묶어 매번 설명 안 해도 되게 | Projects / Projects·GPTs / Gems |
| **장기기억** | 나에 대한 사실을 누적해 다음 대화에 반영 | Memory(제한적) / Memory / 약함 |
| **CLI·데스크탑** | 브라우저 밖에서 — 코드·파일·자동화 | Claude Code·Desktop / Desktop·Codex CLI / 앱 |
| **사이드 캔버스** | 옆 패널에 코드·문서 띄워 반복 수정 | Artifacts / Canvas / Canvas |
| **시스템 지시** | 매 대화에 자동 적용되는 내 규칙 | Project 지시 / Custom Instructions / Custom Gem |
| **도구·데이터 연결** | 캘린더·드라이브·API 등 외부 연결 | MCP(표준) / GPT Actions / Spark |

> 2026년 상반기 기준이에요. 6개월마다 바뀌니 자세한 최신 흐름은 [특강 — 프런티어 AI 지형 2026](/specials/frontier-ai-landscape-2026-05)에서 확인하세요.

처음 셋(작업공간·기억·시스템 지시)은 **지금 당장** 손에 익히면 좋고, 뒤 셋(CLI·캔버스·도구 연결)은 이름만 알아 두면 충분해요. CLI와 도구 연결은 Stage 4·5에서 본격적으로 다룹니다.

딱 하나 주의할 게 있어요. **장기기억(Memory)이 켜진 채로 민감 정보를 던지지 마세요.** 주민번호, 고객 이름, 내부 프로젝트 코드명 같은 게 다음 대화에 슬쩍 새어 나올 수 있거든요. 가끔 들어가서 무엇이 저장됐는지 확인하고 지우는 습관이면 충분해요.

## 그래서, 하나를 정하고 열어 보자

도구를 머리로만 비교하면 끝이 없어요. 오늘은 **매일 쓸 메인 도구 하나를 정하고**, 그 안의 6기능이 어디 있는지 직접 찾아 Project 하나까지 만들어 보는 게 전부예요. 한 번 해 두면, 새 도구가 나와도 "어느 갈래지, 6기능은 어디 있지" 두 가지만 물으면 5분이면 적응합니다.
