## 1. 우리 팀 저항 지도

팀원의 저항을 세 유형으로 분류하고, 각각 다른 대응을 적는다.

| 팀원 (또는 유형) | 저항 표현 | 분류 (불안/불신/불편) | 맞는 대응 |
| --- | --- | --- | --- |
| 예: A | "내 일은 AI가 못 해" | 불안 | 대체가 아니라 보조임을 보여주는 사례 |
| 예: B | "틀린 답 주잖아" | 불신 | 검증 루프를 함께 돌려보기 |
| 예: C | "배울 시간이 없어" | 불편 | 그의 업무 하나를 대신 자동화해 보여주기 |
| | | | |

> 불안 → 안심 · 불신 → 검증된 사례 · 불편 → 대신 해주기
