```
(여기에 현재 쓰고 있는 프롬프트를 그대로 붙여넣기)
```

---

## 2. After — 4축으로 분리

### Task
> *AI 가 만들 최종 결과물 한 문장. 동사 + 목적어.*

```
예) 회의록에서 결정된 Action Item 만 추출해, 담당자·기한과 함께 표로 정리하라.
```

### Context
> *AI 가 알아야 할 배경 — 누가 읽나, 어떤 도메인인가, 무엇을 이미 알고 있나.*

```
예)
- 독자: 마케팅 디렉터
- 회사: B2B SaaS, 회의 빈도 주 2회
- 가정: 회의 참석자 이름은 회의록에 명시됨
- 가정: 기한이 누락된 항목은 "미정" 으로 표기
```

### Constraints
> *지켜야 할 제약 — 길이, 톤, 금지·허용 표현.*

```
예)
- 5줄 이내
- 마크다운 표
- 추측·의견 금지 — 회의록에 명시된 것만
- 기한이 "다음 주" 처럼 모호하면 "미정" 으로
```

### Output
> *결과물의 형식·예시. 가능하면 예시 한 줄을 직접 보여줘라.*

````
| 담당자 | Action | 기한 |
| --- | --- | --- |
| 김민지 | 신규 캠페인 카피 v2 | 6/20 |
````
