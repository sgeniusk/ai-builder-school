## 5. Scout Run 프롬프트

```md
지금은 파일을 수정하지 말고 Scout Run 만 해 주세요.

목표.
- 관련 파일 후보를 찾습니다.
- 현재 구현 관습을 요약합니다.
- 변경 위험과 필요한 테스트를 정리합니다.
- 구현 전에 사람 확인이 필요한 질문을 적습니다.

출력 형식.
1. 읽은 파일.
2. 관련 파일 후보.
3. 발견한 관습.
4. 위험 영역.
5. 다음 구현 계획에 넣을 항목.
```
