## 1. 두 모델 유형 비교

| 항목 | 일반 모델 (instruct) | 추론 모델 (reasoning) |
| --- | --- | --- |
| 성격 | 시키는 대로 또박또박 수행 | 답하기 전에 스스로 생각 |
| 잘 통하는 프롬프트 | 단계를 쪼개 준 안내문 | 목표·기준만 준 지시 |
| 피해야 할 것 | "알아서 해줘" 식 백지 | 단계를 일일이 강제하기 |
| 내가 쓰는 모델 예 | (예: GPT-4o, ___) | (예: o1, ___) |

