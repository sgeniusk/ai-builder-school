## 0. 내 맥락 (3문장)
- 직군 / 역할:
- 'AI' 하면 가장 먼저 떠오르던 것:
- AI에 대해 가장 궁금하거나 불안한 점:

## 1. 'AI' 범위 지도
'AI' 하면 떠오르는 걸 가능한 한 많이 적고, 종류별로 묶는다.

| 종류 | 떠오르는 예 |
| --- | --- |
| 골라주는 것 (추천·검색) |  |
| 알아보는 것 (인식·분류) |  |
| 만들어 내는 것 (생성형) |  |
| 스스로 움직이는 것 (로봇·자율주행·에이전트) |  |
