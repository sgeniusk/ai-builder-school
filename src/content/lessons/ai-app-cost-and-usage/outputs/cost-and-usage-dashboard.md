## 2. 사용 시나리오별 월 비용

| 시나리오 | 사용자 수 | 사용자당 일 요청 수 | 요청당 입력 토큰 | 요청당 출력 토큰 | 월 사용 일수 | 월 예상 비용 |
| --- | --- | --- | --- | --- | --- | --- |
| 가벼운 사용 | | | | | | |
| 평균 사용 | | | | | | |
| 많이 쓰는 사용 | | | | | | |

계산 메모.

- 입력 비용 = 사용자 수 × 일 요청 수 × 입력 토큰 × 월 사용 일수 × 입력 단가.
- 출력 비용 = 사용자 수 × 일 요청 수 × 출력 토큰 × 월 사용 일수 × 출력 단가.
- 총 비용 = 입력 비용 + 출력 비용 + 임베딩 비용 + 기타 인프라 비용.
