## 2. 알림·한도 설정 체크리스트

각 종량제 도구마다 콘솔에서 아래를 건다.

- [ ] 사용량 알림 50% 설정
- [ ] 사용량 알림 80% 설정
- [ ] 사용량 알림 100% 설정
- [ ] 지출 한도(hard limit) 설정 — 넘으면 호출이 막히는 금액
- [ ] 알림 수신처(이메일/문자)가 내가 매일 보는 곳인가

## 3. 주간 5분 점검 루틴

- 점검 요일·시간 (캘린더 고정 슬롯):
- 확인 순서
	1. 이번 주 누적 사용량 — 예산 대비 몇 %인가
	2. 지난주 대비 급증한 항목이 있는가
	3. 새로 켠 자동화·스크립트가 있는가
	4. 공유한 API 키 중 회수할 것이 있는가

## 4. 비용 급증 대응 절차

예상의 N배가 찍혔을 때 순서대로.

1. **멈춤** — 의심되는 API 키를 즉시 회수하거나 한도를 0으로
2. **확인** — 사용량 로그에서 언제·어디서 급증했는지 본다
3. **분류** — 원인이 (a) 자동화 폭주 (b) 키 오용 (c) 정상 증가 중 무엇인가
