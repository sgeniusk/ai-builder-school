# AI PR Eval 게이트 가이드

AI 코딩 에이전트가 생성한 코드를 머지 전에 검증하는 체크리스트와 LLM 리뷰 프롬프트.

## 이 폴더의 파일

| 파일 | 용도 |
|------|------|
| `pr-checklist.md` | AI 코드 전용 PR 머지 전 체크리스트 |
| `llm-review-prompt.md` | AI가 AI 코드를 리뷰하는 프롬프트 |

## 사용 순서

1. AI 코딩 에이전트가 PR을 생성하면 `pr-checklist.md` 실행
2. 체크리스트 통과 후 `llm-review-prompt.md`로 LLM 2차 리뷰 요청
3. 두 단계 모두 통과하면 머지

## 핵심

"잘 됐나요?"가 아니라 구체적인 오류 패턴을 지목해야 실질적인 검증이 된다.
