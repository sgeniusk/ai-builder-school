## API 키가 새면 청구서가 온다

Claude API를 며칠 써봤는데 한 달 후 이메일이 왔다. "$47 청구됩니다."

두 가지 실수였다. 첫째, API 키를 GitHub에 올렸다. 봇이 자동으로 수집해 다른 사람이 내 키로 요청을 보냈다. 둘째, spending limit을 설정하지 않았다.

이 레슨은 이 두 가지 실수를 사전에 막는다.

---

## 필요한 계정 목록

| 계정 | 용도 | 무료 여부 |
|------|------|---------|
| GitHub | 코드 저장소, 협업 | 무료 (공개 저장소) |
| Anthropic Console | Claude API 키 발급 | API 사용량 기반 과금 |
| OpenAI Platform | GPT API 키 발급 (선택) | API 사용량 기반 과금 |

**Claude.ai 웹 vs Anthropic API는 다르다.** Claude.ai 웹($20/월 Pro)은 브라우저에서 쓰는 것이고, Anthropic API는 코드에서 호출하는 것이다. 이 학교 실습에서는 API를 쓴다.

---

## API 키 안전하게 저장하기

**절대 이렇게 하지 않는다:**

```python
# ❌ 코드 안에 직접 쓰기 — GitHub에 올라가는 순간 봇이 가져간다
client = anthropic.Anthropic(api_key="sk-ant-api03-...")
```

**이렇게 한다:**

### 1단계 — .env 파일 만들기

```bash
# my-ai-builder-lab 폴더에서
touch .env
```

`.env` 파일에 입력:

```
ANTHROPIC_API_KEY=sk-ant-api03-여기에-실제-키-입력
```

### 2단계 — .gitignore에 추가

```bash
touch .gitignore
echo '.env' >> .gitignore
```

`.gitignore`에 `.env`가 있으면 Git이 해당 파일을 무시한다. GitHub에 올라가지 않는다.

### 3단계 — 코드에서 불러오기

```python
import os
from dotenv import load_dotenv

load_dotenv()
api_key = os.environ.get("ANTHROPIC_API_KEY")
```

이 패턴이 표준이다. 한 번 익혀두면 모든 프로젝트에 동일하게 쓴다.

## 비용 구조 이해

AI API는 **토큰** 단위로 과금된다. 토큰은 대략 단어 0.75개에 해당한다.

```
Claude 3.5 Sonnet 기준 (2025년 5월):
- 입력: $3 / 1M 토큰
- 출력: $15 / 1M 토큰

실용적으로:
- 1,000단어 입력 + 500단어 출력 ≈ 약 $0.003 (0.3센트)
- 하루 100회 호출 ≈ 약 $0.30
```

개발·실습 중에는 비용이 거의 안 나온다. 하지만 루프를 잘못 짜서 API를 수천 번 호출하면 금방 오른다.

## Spending Limit 설정

Anthropic Console에서 반드시 설정한다.

1. [console.anthropic.com](https://console.anthropic.com) 로그인
2. Settings → Limits
3. **Monthly spend limit** → $5 또는 $10으로 설정

한도에 도달하면 API 호출이 중단된다. 학습·실습 중에는 $5로도 충분하다.
