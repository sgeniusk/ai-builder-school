## 결제 카드는 아직 꺼내지 않는다

AI 도구를 처음 켜면 곧바로 "월 $20 구독" "결제 카드 등록" 화면이 나온다. 여기서 많은 사람이 멈춘다. 카드를 등록할지 말지 정하느라 학습이 시작도 못 한 채 끝난다.

좋은 소식이 있다. **이 학교의 Stage 1~4는 결제 없이, 무료 한도만으로 따라올 수 있다.** 결제가 정말 필요한 건 Stage 5부터다 — AI를 *코드에서* 호출하기 시작할 때.

그러니 길을 두 갈래로 나눈다.

| 트랙 | 누구를 위한 것 | 무엇으로 | 결제 |
|------|--------------|---------|------|
| **무료 한도 트랙** | 지금 — Stage 1~4를 가볍게 시작 | Claude·ChatGPT·Gemini 무료 웹 | 카드 등록 없음 |
| **유료 API 트랙** | Stage 5부터 — 코드에서 AI 호출 | API 키 + `.env` | 사용량 기반 (월 $5부터) |

지금 할 일은 **무료 한도 트랙**뿐이다. 유료 API 트랙은 Stage 5에 도착했을 때 이 레슨으로 다시 돌아와서 하면 된다.

## 유료 API 트랙 — Stage 5에 도착하면

Stage 5부터는 AI를 채팅창이 아니라 *코드에서* 호출한다. 그때 **API 키**가 필요하고, API는 쓴 만큼 돈이 나간다. 아래는 Stage 5에 도착했을 때 이 레슨으로 돌아와서 하면 된다. 지금 미리 해두고 싶다면 해도 좋다.

### Claude.ai 웹 vs Anthropic API는 다르다

Claude.ai 웹(무료 한도 / Pro $20월)은 브라우저에서 쓰는 것이고, Anthropic API는 코드에서 호출하는 것이다. 무료 한도 트랙은 웹을 쓰고, 유료 API 트랙은 API를 쓴다.

### API 키 안전하게 저장하기

API 키가 새면 청구서가 온다. 키를 GitHub에 올리면 봇이 자동으로 수집해 다른 사람이 내 키로 요청을 보낸다.

**절대 이렇게 하지 않는다:**

```python
# ❌ 코드 안에 직접 쓰기 — GitHub에 올라가는 순간 봇이 가져간다
client = anthropic.Anthropic(api_key="sk-ant-api03-...")
```

**이렇게 한다:**

```bash
# my-ai-builder-lab 폴더에서
touch .env
touch .gitignore
echo '.env' >> .gitignore
```

`.env` 파일에 입력:

```
ANTHROPIC_API_KEY=sk-ant-api03-여기에-실제-키-입력
```

`.gitignore`에 `.env`가 있으면 Git이 그 파일을 무시한다. GitHub에 올라가지 않는다. 코드에서는 이렇게 불러온다:

```python
import os
from dotenv import load_dotenv

load_dotenv()
api_key = os.environ.get("ANTHROPIC_API_KEY")
```

이 패턴이 표준이다. 한 번 익혀두면 모든 프로젝트에 똑같이 쓴다.

### Spending Limit 설정

유료 API를 쓰기로 했다면 한도를 반드시 건다.

1. [console.anthropic.com](https://console.anthropic.com) 로그인
2. Settings → Limits
3. **Monthly spend limit** → $5 또는 $10으로 설정

한도에 도달하면 API 호출이 중단된다. 학습·실습 중에는 $5로도 충분하다.

### 비용 구조 이해

AI API는 **토큰** 단위로 과금된다. 토큰은 대략 단어 0.75개에 해당한다.

```
Claude 3.5 Sonnet 기준:
- 입력: $3 / 1M 토큰
- 출력: $15 / 1M 토큰

실용적으로:
- 1,000단어 입력 + 500단어 출력 ≈ 약 $0.003 (0.3센트)
- 하루 100회 호출 ≈ 약 $0.30
```

개발·실습 중에는 비용이 거의 안 나온다. 하지만 루프를 잘못 짜서 API를 수천 번 호출하면 금방 오른다. 그래서 spending limit이 안전망이다.
