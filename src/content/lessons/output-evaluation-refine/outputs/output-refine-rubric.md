> 이 파일은 AI Builder School · Stage 2 "AI 출력 평가하고 다듬기" 에서 제공되는 출발 템플릿입니다.
> AI 출력 하나를 골라 채점하고, refine을 한 바퀴 돌리며 빈칸을 채우세요.

## 1. 채점 항목 5개

각 항목을 1~3점으로 매긴다 (1 = 나쁨, 2 = 보통, 3 = 좋음).

| 항목 | "좋음(3점)" 기준 | 점수 |
| --- | --- | --- |
| 정확성 | 사실·숫자·인용이 모두 검증되거나 출처가 분명함 | / 3 |
| 완결성 | 요청한 항목이 하나도 빠지지 않음 | / 3 |
| 톤 | 받는 사람·상황에 맞는 말투임 (예: 사내 보고 = 간결·중립) | / 3 |
| 구조 | 제목·문단·목록이 읽기 좋게 배열됨 | / 3 |
| 길이 | 핵심을 담되 늘어지지 않음 | / 3 |

## 3. 수정 지시 문장 틀 3개

1. "[항목]을 — [위치]를 — [방향]으로 고쳐줘."
	<!-- 예: 톤을 — 인사말 문단을 — 더 격식 있게 고쳐줘. -->
2. "[위치]에 [빠진 내용]을 추가해줘."
	<!-- 예: 결론부에 다음 행동 항목 3개를 추가해줘. -->
