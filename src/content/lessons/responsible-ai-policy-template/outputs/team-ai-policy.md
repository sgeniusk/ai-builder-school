## 1. 자유롭게 사용해도 되는 것

- 내부 아이디어 탐색, 브레인스토밍
- 초안 작성 (초안이지, 최종본 아님)
- 코드 리뷰 보조, 테스트 케이스 생성
- 개인 학습, 정보 탐색
- 내부 문서 요약 (개인정보 제외)

---

## 2. 필수 검토 후 사용

AI가 만든 결과물을 그대로 쓰지 않는다. 사람이 검토한다.

**고객 응대 메시지**
→ 발송 전 반드시 읽어볼 것. 틀린 정보, 어색한 톤 확인.

**마케팅·홍보 콘텐츠**
→ 사실 확인 + 브랜드 톤 일치 여부 확인 후 발행.

**분석 보고서**
→ 수치가 원본 데이터와 일치하는지, 인과관계 주장에 근거가 있는지 확인.

**채용·인사 관련 문서**
→ AI 편향 가능성 인식. 최종 판단은 사람이.

---

## 3. 금지

**개인정보**
고객 이름, 이메일, 연락처, 주민번호, 결제 정보를 외부 AI 서비스에 입력하지 않는다.
익명화하거나 가상 데이터로 대체할 것.

**법적 효력이 있는 문서**
계약서, 법적 의견서를 AI 출력 그대로 서명하거나 발행하지 않는다.
법무 검토 필수.

**AI 출력의 무검토 발행**
고객·외부에 나가는 모든 결과물은 사람이 최종 확인한다.

## 4. AI 표시 의무

- **외부 발행 콘텐츠:** 블로그, SNS, 보도자료는 AI 보조 작성 사실을 필요 시 명시
- **제안서·보고서:** 클라이언트 요청 시 AI 보조 사용 여부 고지
- **팀 내부 문서:** 표시 의무 없음
