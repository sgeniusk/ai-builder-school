# 팀 AI 정책 가이드

금지/자유사용/필수검토 3레이어 기반 팀 AI 사용 정책과 신규 팀원 온보딩 체크리스트.

## 이 폴더의 파일

| 파일 | 용도 |
|------|------|
| `team-ai-policy.md` | 팀 AI 사용 정책 (1~2페이지, 가이드 톤) |
| `onboarding-checklist.md` | 신규 팀원 AI 정책 이해 확인 체크리스트 |

## 사용 순서

1. `team-ai-policy.md` 의 `[팀 정보]` 칸을 팀 상황에 맞게 편집
2. 팀 미팅에서 정책 공유 및 피드백 수렴
3. 신규 팀원 합류 시 `onboarding-checklist.md` 로 이해도 확인
4. 분기 1회 정책 검토 및 버전 업데이트

## 핵심

금지 목록보다 "이럴 땐 이렇게" 가이드 중심으로 써야 팀원이 실제로 읽는다.
