# 프롬프트 디버그 노트

> 이 파일은 AI Builder School · Stage 2 "프롬프트가 실패할 때 — 4축 디버깅 루프" 에서 제공되는 출발 템플릿입니다.
> 프롬프트가 실패할 때마다 통째로 다시 쓰지 말고, 여기에 진단을 쌓으세요.

## 1. 증상 → 축 매핑표

실패의 모양을 보고 어느 축이 약했는지 진단한다.

| 증상 | 약한 축 | 고칠 방향 |
| --- | --- | --- |
| 답이 엉뚱한 방향이다 | Task | AI가 할 일을 한 문장으로 더 명확하게 |
| 사실·숫자가 틀렸다 | Context | 필요한 배경 정보·자료를 프롬프트에 넣기 |
| 너무 길거나 범위를 벗어났다 | Constraints | 분량·범위·금지사항을 명시 |
| 형식이 깨졌다 | Output | 출력 형식을 예시와 함께 지시 |
