# A/B 테스트 의사결정 메모 가이드

통계 수치를 의사결정 행동으로 바꾸는 워크플로. 낙관적 해석 편향을 방어하고 명확한 결론을 도출한다.

## 이 폴더의 파일

| 파일 | 용도 |
|------|------|
| `decision-memo-template.md` | 5단 의사결정 메모 템플릿 |
| `interpretation-prompt.md` | 제약 사항 우선 해석 프롬프트 |

## 사용 순서

1. `interpretation-prompt.md` 로 AI에게 제약 사항 먼저 물어보기
2. AI 응답 검토 후 `decision-memo-template.md` 수치 입력
3. 결론(배포/보류/재실험) 작성
4. 다음 행동 3가지 구체화
5. 이해관계자 공유

## 핵심

결론은 반드시 하나. 배포 / 보류 / 재실험 중 하나를 선택한다. 애매한 결론은 결론이 아니다.
