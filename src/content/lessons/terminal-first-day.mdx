## 터미널은 무섭지 않다

검은 화면에 커서만 깜빡인다. 뭘 쳐야 할지 모른다. 잘못 입력하면 컴퓨터가 망가질 것 같다.

전부 오해다.

터미널은 컴퓨터에게 텍스트로 명령하는 창일 뿐이다. 잘못 입력해도 `Ctrl+C`로 멈추거나 `cd ~`로 홈으로 돌아오면 된다. 파일을 실수로 삭제하지 않는 한 터미널 명령어로 컴퓨터가 망가지지 않는다.

---

## 터미널 열기

**Mac:** Spotlight(`Cmd+Space`) → "Terminal" 검색 → Enter

**Windows:** 시작 메뉴 → "PowerShell" 검색 → Enter
(또는 Git Bash를 설치하면 Mac과 동일한 명령어를 쓸 수 있다)

---

## 생존 명령어 10개

### 1. 현재 위치 확인

```bash
pwd
```

"Print Working Directory". 지금 어느 폴더에 있는지 보여준다. 길을 잃으면 가장 먼저 친다.

### 2. 폴더 목록 보기

```bash
ls          # Mac / Linux
dir         # Windows PowerShell
```

현재 폴더 안에 무엇이 있는지 보여준다.

```bash
ls -la      # 숨김 파일 포함, 상세 정보
```

### 3. 폴더 이동

```bash
cd Documents
cd my-ai-builder-lab
```

"Change Directory". 폴더 이름을 Tab 키로 자동완성할 수 있다.

### 4. 상위 폴더로

```bash
cd ..
```

점 두 개(`..`)는 '한 단계 위'를 의미한다.

### 5. 홈 디렉토리로

```bash
cd ~
```

어디서든 홈(내 사용자 폴더)으로 돌아온다. 길을 잃었을 때.

### 6. 새 폴더 만들기

```bash
mkdir my-ai-builder-lab
```

"Make Directory". 폴더를 만든다.

### 7. 빈 파일 만들기

```bash
touch README.md          # Mac / Linux
new-item README.md       # Windows PowerShell
```

### 8. 파일 내용 보기

```bash
cat README.md
```

파일 내용을 터미널에 출력한다.

### 9. VS Code로 현재 폴더 열기

```bash
code .
```

점(`.`)은 '현재 폴더'를 의미한다. VS Code가 설치되어 있어야 한다.

### 10. 명령 중단

```bash
Ctrl+C
```

실행 중인 명령을 강제로 멈춘다. 뭔가 잘못됐을 때 제일 먼저 시도한다.

## 실습 순서

터미널을 열고 아래를 순서대로 입력한다.

```bash
# 1. 지금 어디에 있는지 확인
pwd

# 2. 홈으로 이동
cd ~

# 3. 폴더 목록 확인
ls

# 4. 실습 폴더 생성
mkdir my-ai-builder-lab

# 5. 폴더로 이동
cd my-ai-builder-lab

# 6. README 파일 생성
touch README.md

# 7. 파일 목록 확인 (README.md가 보이면 성공)
ls -la
```

## 에러가 났을 때

터미널 에러는 정보다. 무서워하지 말고 읽는다.

**가장 흔한 에러:**

```
No such file or directory
```
→ 폴더 이름이 틀렸거나 대소문자가 다른 것. `ls`로 폴더 이름을 확인하고 다시 시도.

```
command not found
```
→ 해당 도구가 설치 안 됐거나 이름이 틀린 것.

**해결 방법:** 에러 메시지를 AI에게 통째로 붙여넣는다.

```
이 에러가 떴어요:
No such file or directory: 'Documents/my-project'
Mac 터미널에서 cd 명령어를 쓰고 있어요. 어떻게 해결하나요?
```

이 방법으로 거의 모든 터미널 에러가 해결된다.

## 기억할 것

| 단축키 | 기능 |
|--------|------|
| `Tab` | 폴더·파일 이름 자동완성 |
| `↑` (위 화살표) | 이전에 입력한 명령어 |
| `Ctrl+C` | 실행 중인 명령 중단 |
| `Ctrl+L` | 화면 지우기 |

`Tab` 자동완성을 쓰면 오타가 줄어든다. 폴더 이름 첫 몇 글자만 치고 Tab을 누른다.
