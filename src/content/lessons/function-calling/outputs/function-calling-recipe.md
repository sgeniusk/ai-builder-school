## 2. Tool Schema 초안

```ts
const tool = {
  name: "getWeather",
  description: "도시 이름을 받아 현재 날씨 요약을 반환한다.",
  parameters: {
    type: "object",
    properties: {
      city: {
        type: "string",
        description: "날씨를 조회할 도시 이름"
      },
      unit: {
        type: "string",
        enum: ["celsius", "fahrenheit"]
      }
    },
    required: ["city", "unit"]
  }
};
```

내 함수에 맞게 바꿀 값.

| 항목 | 결정 |
| --- | --- |
| 함수 이름 | |
| 한 문장 설명 | |
| 필수 인자 | |
| 선택 인자 | |
| enum으로 제한할 값 | |
