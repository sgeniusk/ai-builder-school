## 내가 집중할 Phase

아래 Phase 중 내 목표와 가장 관련 있는 것을 체크한다.

- [ ] Phase 1 — AI 리터러시 (AI를 비판적으로 쓰는 법)
- [ ] Phase 2 — 프롬프트 엔지니어링 (AI에게 정확히 지시하는 법)
- [ ] Phase 3 — 일상 자동화 (반복 업무 자동화)
- [ ] Phase 4 — 코딩 에이전트 (AI와 함께 코드 만들기)
- [ ] Phase 5 — AI 앱 개발 (API 연결 서비스)
- [ ] Phase 6 — RAG (내 문서로 AI 강화)
- [ ] Phase 7 — 에이전트 & MCP (자율 동작 에이전트)
- [ ] Phase 8 — 멀티모달 & 콘텐츠 (영상·이미지·글 자동화)
- [ ] Phase 9 — 데이터 분석 (숫자를 판단으로)
- [ ] Phase 10 — Evals & 보안 (안전한 AI 운영)
- [ ] Phase 12 — 캡스톤 (실제 프로젝트 완성)
