## 2. 5축 60초 질문 세트

### F · Fact (사실)
- [ ] 모든 수치가 원본 자료와 일치하는가?
- [ ] 인명·기관명·제품명이 정확한가?
- [ ] 인용된 문장이 원문 그대로인가?

### O · Origin (출처)
- [ ] 각 주장의 출처가 명시되어 있는가?
- [ ] 출처 링크가 실제로 그 내용을 담고 있는가? (Lesson 1.3 verify-loop 호출)
- [ ] 모델 추정과 검증된 사실이 구분되어 있는가?

### S · Scope (범위)
- [ ] 답이 내 질문 범위를 넘지 않았는가?
- [ ] 일반화가 너무 광범위한가? ("모든...", "항상...", "절대...")
- [ ] 빠진 예외 시나리오는 없는가?

### C · Confidence (신뢰)
- [ ] 단정 어조가 정당한가? (낮은 신뢰도 + 단정 = 환각 신호)
- [ ] 모델이 신뢰도 점수를 적었는가? 60 이하 주장은 표시되어 있는가?
- [ ] 내 강·약점 지도(Lesson 1.2)에서 "약함"으로 분류된 영역인가?

### B · Bias (편향)
- [ ] 한쪽 입장만 강조되었는가?
- [ ] 부정적·예외적 시나리오가 누락되었는가?
- [ ] 학습 데이터의 시기·문화 편향이 들어왔을 가능성은?
