## 3. 스텝 규칙

| 단계 | 에이전트가 하는 일 | 로그에 남길 것 |
| --- | --- | --- |
| Think | 다음 행동 선택 | 선택 이유 |
| Act | 도구 호출 | 도구 이름, 인자 요약 |
| Observe | 결과 확인 | 성공 여부, 핵심 결과 |
| Review | 완료 또는 다음 스텝 결정 | 종료 판단 |

---

## 4. 종료 조건

| 조건 | 처리 |
| --- | --- |
| 목표 달성 | 최종 답변 반환 |
| 최대 스텝 도달 | 중단 요약과 사람 확인 요청 |
| 같은 도구와 인자 반복 실패 | 루프 중단 |
| 권한 필요한 도구 요청 | 확인 단계로 전환 |
| 도구 결과가 비어 있음 | 질문 재작성 또는 중단 |

## 5. 관찰 로그 예시
