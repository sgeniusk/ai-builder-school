# PM Eval 가이드

코드 없이 AI 출력 품질을 측정하는 Eval 루브릭과 개발팀 전달용 기준 정의서.

## 이 폴더의 파일

| 파일 | 용도 |
|------|------|
| `eval-rubric.md` | 비즈니스 목표 기반 측정 가능한 Eval 루브릭 |
| `quality-criteria.md` | 개발팀 전달용 품질 기준 정의서 |

## 사용 순서

1. `eval-rubric.md` 의 `[비즈니스 목표]` 칸을 채워 루브릭 커스터마이즈
2. AI 출력 샘플 10개를 스프레드시트에 넣고 루브릭으로 채점
3. 평균 점수 계산 → 현재 품질 수준 파악
4. `quality-criteria.md` 를 개발팀과 공유 → 자동화 기준으로 활용
5. 주간 루틴: 매주 10개 샘플 채점, 점수 트렌드 기록

## 핵심

두 사람이 같은 샘플을 채점했을 때 같은 결과가 나와야 좋은 기준이다.
