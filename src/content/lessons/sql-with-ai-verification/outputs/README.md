# SQL AI 워크플로 가이드

AI로 SQL 쿼리를 작성하고 검증하는 3단계 루프.

## 이 폴더의 파일

| 파일 | 용도 |
|------|------|
| `schema-notes.md` | 자주 쓰는 테이블 스키마 메모 |
| `query-prompt-template.md` | 스키마 주입 + 셀프 리뷰 재사용 프롬프트 |

## 사용 순서

1. `schema-notes.md` 에 자주 쓰는 테이블 스키마 업데이트
2. `query-prompt-template.md` 의 `[스키마]` 칸에 붙여넣기
3. `[요청]` 칸에 원하는 분석 작성
4. AI 응답 쿼리를 셀프 리뷰 프롬프트로 검토
5. 5행 테스트 데이터로 결과 확인 후 실행

## 핵심 원칙

스키마 없이 요청하지 않는다. AI가 컬럼명을 지어낸다.
