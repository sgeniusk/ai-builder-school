# Future Roadmap — AI Builder School

> MVP 이후의 반복(iteration) 계획. 우선순위는 학습자 피드백에 따라 조정한다.

## v0.2 — 콘텐츠 심화 & 편집 개선
- MDX 전환: 장문 본문, 코드 블럭, 이미지 삽입 지원.
- 레슨 검색 + 태그 필터.
- 코드 예시 하이라이팅 (Shiki).
- Open Graph 이미지 자동 생성.
- Vitest: 콘텐츠 무결성 유닛 테스트.

## v0.3 — 학습자 경험
- 진도 저장 (localStorage 기반).
- 체크리스트 상호작용 (스스로 체크 → 진행률 표시).
- 레슨별 "내 산출물 링크" 붙이기.
- 학습자 페르소나 기반 홈 개인화.

## v0.4 — 커뮤니티 & 공유
- GitHub Discussions 임베드.
- 학습자 쇼케이스 갤러리 (CMS 또는 JSON 수동 관리).
- "이 주의 미션" 피드.

## v0.5 — 실습 샌드박스
- Next.js Route Handlers로 무료 LLM API 샘플 실습.
- Stackblitz/WebContainers 임베드 실험.
- Claude Code 플러그인으로 레슨 import.

## v0.6 — 다국어
- i18n 라우팅 (`/en`, `/ja`).
- `titleKo` / `titleEn` 외 본문 번역 파이프라인.

## v0.7 — 제품화 지원
- 유료 트랙 / 프리미엄 프로젝트 (Stripe Checkout).
- 기업용 워크숍 파일 세트 (PDF/Notion).

## 장기 아이디어
- "AI 사용 일지" 모듈: 학습자가 매일 AI와의 판단 기록 남김.
- 조직 배포: 커스터마이즈된 내부 커리큘럼.
- 평가/인증 시스템 (자동 채점 + 휴먼 리뷰).
- 파트너 도구(OpenAI, Anthropic, Cursor, Vercel)와의 공식 샘플.

## 하지 않을 것 (Non-goals)
- 일반 AI 뉴스 블로그.
- 강의 판매 중심 LMS.
- 실시간 채팅/DM 커뮤니티 운영.
- 과장된 "30일 AI 전문가" 마케팅.
