## 4. 막힌 지점 · 이탈 지점 (구체 인용)
첫째, L01의 "터미널 (Terminal)" 정의 "마우스 클릭 대신 글자로 명령한다"는 표현이 영업 직군에 와닿지 않는다. "의사가 MRI 기계 원리를 몰라도 영상을 보고 판단하듯이"라는 비유는 좋은데, 정작 그다음 미션이 터미널을 열어 명령어를 치는 것이라 비유와 미션이 어긋난다. "AI를 쓰려고 들어왔는데 왜 검은 화면부터?"가 가장 강한 의문이었다.
둘째, L02 "Mac: Spotlight(Cmd+Space) → 'Terminal' 검색"이 한 줄로 끝나고, 회사 지급 노트북이 Windows인 사람을 위한 분기 설명이 약하다. PowerShell·Git Bash를 두고 결정도 못 하는데 본문은 다음 명령어로 넘어간다. 모바일에서 보면 코드 블록이 가로로 잘려서 명령어 끝이 안 보인다.
셋째, L03 미션의 "GitHub, Claude/OpenAI API, Codex CLI"를 모두 깔라는 지시가 너무 무겁다. 영업 직군에게 GitHub 계정은 평생 만들 일이 없었던 것이고, OpenAI API 결제 카드 등록은 "이걸 굳이?" 하고 앱을 닫게 만든다. ChatGPT 음성 메모를 더 잘 쓰려고 들어왔는데, 그 길로 안내해주지 않는다.
