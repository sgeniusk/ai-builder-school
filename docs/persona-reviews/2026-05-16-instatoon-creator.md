# P08 박나래 — 인스타툰 작가가 검은 창 앞에서 멈춘 이유

## 1. 페르소나 프로필

29세, 인천 거주, 인스타툰 팔로워 1.2만. 본업은 작가, 부업은 굿즈 판매. Procreate·Clip Studio를 매일 쓰고, 최근 6개월간 ChatGPT 무료 버전과 Midjourney로 캐릭터 시안과 콘티 빠르게 뽑는 데 익숙해짐. 코드는 평생 안 배울 작정. 사이트에 들어온 이유는 "Claude로 화풍 일관성 있는 캐릭터 시안 더 잘 뽑고 싶음" 한 줄. 모바일 위주, 출근길 지하철과 점심시간 합쳐 하루 40분. 의심은 낮아 일단 따라해 보는 편인데, 그래서 더 멀리 갔다가 더 크게 실망한다.

## 4. 막힌 지점 인용

> L01 zero-coding-orientation의 "터미널이 왜 필요한가" — 그림 그리는 사람한테 왜 검은 창이 필요해요? 저는 Procreate 켜고 펜 잡으면 끝이에요. AI로 캐릭터 시안 빨리 뽑고 싶은 게 전부인데, 첫 lesson부터 다른 직업군 얘기 같았어요.

> L03 ai-tool-account-and-cost의 "GitHub, Claude/OpenAI API, Codex CLI" — 인스타툰 작가는 GitHub 자체가 없어요. API 키 발급 화면 들어갔다가 결제 카드 등록하라는데, "이게 한 달에 얼마인지" 가늠이 안 돼서 그냥 닫았어요. Claude 유료 결제는 이미 하고 있는데 그것만 활용할 길은 사이트에 안 보였어요.

> Stage 4·4b의 추천 도구 박스 "Midjourney·Stable Diffusion" — Midjourney는 매일 쓰는데 Stable Diffusion은 들어본 적도 없어요. Mac M1 16GB라 로컬 모델 깔라고 하면 노트북 팬 돌아갈까봐 무서워요. "Midjourney만 쓰는 트랙"이라는 갈래가 있으면 좋겠어요.

> Stage 1 L02 "검은 화면에 커서만 깜빡인다" — 이걸 읽고 갸웃해서 진짜 터미널을 켰는데, M1에서 권한 알림이 뜨고 뭘 허용해야 할지 몰라서 그냥 꺼버렸어요. 작가용 분기 안내가 한 줄도 없었어요.

## 8. 종합 의견

랜딩 카피와 Stage 4 도입부의 "동료로 둡니다"는 인스타툰 작가 감성에 잘 맞다. 실제로 Stage 4·4b의 두 lesson은 본업 도구를 1단계 업그레이드시켜 줬다 — 화풍 일관성 프롬프트 시스템과 굿즈샵 콘텐츠 캘린더. 문제는 사다리의 첫 두 칸이다. Creator 여정으로 들어왔는데도 Stage 1 L01·L02·L03이 코딩 에이전트·터미널·API 키·과금 세팅으로 빼곡해서, 그림 그리는 사람이 첫날부터 "내 진영이 아니다"라는 신호를 받는다. 의심이 낮은 학습자이기에 첫 일주일은 끈기로 버텼지만, 결국 사이드바 미리보기로 슬쩍 점프해서 본인 진영 lesson 2개를 채우고 끝났다. NPS 6은 "Stage 4 이후만 보면 8, Stage 1만 보면 3"의 평균값이다. Creator 여정 입구를 Stage 4 어딘가로 옮기거나, Stage 1의 Creator 분기 안내를 명시적으로 깔아야 한다.

## 9. 개선 제안 3개

1. Creator 여정 전용 Stage 1 분기 — "그림 작가·영상 크리에이터는 Stage 1 L02·L03을 건너뛰고 Stage 4·4b로 바로 가도 됩니다" 같은 분기 안내를 L01 본문 상단에 박는다. 추후 필요해질 때 돌아오는 길도 함께 표기한다.
2. Midjourney 단일 트랙 lesson 신설 — `creator-mj-only-track` 슬러그로, Stable Diffusion 없이도 화풍 일관성·캐릭터 시트·콘티 빠르게 뽑는 워크플로를 1개 lesson으로 묶는다. Mac M1 환경 가이드도 함께 포함한다.
3. 진척 표시 재정의 — "사다리 5%"가 아니라 "Creator 트랙 진행 X%"로 본인 여정 안 진척을 우선 노출. 사이드바 미리보기 점프에서 시작한 학습 경로도 가중치에 정상 반영해 의욕 손실을 줄인다.
