## 종합 의견

학교는 "AI를 진지하게 처음 배우는 사람"을 Explorer 여정에 명시했지만, 첫 lesson의 톤은 결국 "AI로 코드를 만드는 사람"을 가정한다. 28년차 간호사가 들어와 첫 카드에서 "터미널·코드·과금"이라는 단어 3개를 동시에 맞는다. 학교의 비전 카피 "AI를 쓰는 사람에서, AI로 만드는 사람으로"는 본인을 끌어당겼지만, Stage 1의 첫 세 lesson은 "이미 쓰는 사람"을 전제로 짜여 있다. 입문자가 "쓰는 사람" 단계로 들어오는 lesson이 사실상 없는 셈이다. NPS 3, 콘텐츠 3.0, UI 2.6은 콘텐츠 자체의 질이 낮다는 평가가 아니라, 본인 같은 입문자에게 도달이 되지 않는다는 평가다. 손주 추천이 아니었다면 D2에서 종료됐을 흐름. 그래도 도입 카피 "AI는 당신의 능력을 빼앗지 않습니다"는 본인 직업관과 닿아 한 줄짜리 호감을 남겼다.

## 개선 제안 3개

1. Stage 1 앞에 "Stage 0 · 사용자로 만나기" 마이크로 사다리를 신설, 코드·터미널·API 없이 ChatGPT/Claude 사용자 경험만 다루는 3 lesson을 두고 비개발 직군(간호·자영업·은퇴자) 시나리오를 직접 노출.
2. 모바일에서 코드 블록을 카드형으로 변환(가로 스크롤 제거), "큰 글씨 모드"와 "전문용어 한 줄 풀이" 토글을 사이드바 상단에 둔다.
3. `ai-tool-account-and-cost`를 3개 mini-step(GitHub만, API 키만, CLI만)으로 분할하고, 결제 카드가 없는 사용자용 무료 한도/체험 키 분기를 카드 단위로 노출. 첫 도전에서 "결제 카드 단계"를 만나지 않도록 한다.
